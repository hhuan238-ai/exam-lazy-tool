# 考試懶人工具

免安裝的靜態網頁應用，可儲存公式或模型，貼上或上傳資料表後批次產生分析欄位。

## 開啟

直接用瀏覽器開啟 `index.html`。

## 公式範例

```js
[期中] * 0.35 + [期末] * 0.45 + [作業] * 0.2
[總分] >= 60 ? "及格" : "補考"
round(([分數] - avg("分數")) / sd("分數"), 2)
rank("分數", "desc")
```

## 資料格式

支援貼上 CSV、TSV、分號分隔資料，也支援上傳 `.csv`、`.tsv`、`.txt`、`.xlsx`、`.xls`。

模型會存在瀏覽器的 localStorage，不會上傳到伺服器。
